  NEXT STEPS FOR UBUNTU DEPLOYMENT
================================================================================

On your Ubuntu machine (home-server):

1. Pull the latest code:
   $ cd ~/fantasy_basketball_tools
   $ git pull

2. Create configuration:
   $ cp .env.example .env
   $ nano .env
   
   Fill in your ESPN credentials:
   - ESPN_S2: [from browser cookies]
   - SWID: [from browser cookies]
   - LEAGUE_ID: [your league ID]

3. Start the service:
   $ docker-compose up -d

4. Verify it's running:
   $ docker-compose ps
   $ curl http://localhost:8000/api/v1/health

5. View logs:
   $ docker-compose logs -f
